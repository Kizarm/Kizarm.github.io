Ovládání je pouze pomocí klávesnice

  • Start a Stop animace pomocí klávesy Space (mezerník)
  • Rychlost animace se nastaví numerickými klávesami 0 až 9, default střední rychlost je 5
  • Escape smaže obrazovku
  • Enter přepíná typ animace

Sice to není k ničemu, ale hezky to ukazuje, jak náhodně vygenerovaná čísla, když jsou trochu usměrněna jednoduchou matematikou, mohou vytvářet složité obrazce. Pokud chcete vědět, jak to funguje, podívejte se do zdrojáků.
