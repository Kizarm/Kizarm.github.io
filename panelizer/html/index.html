Postscript Panelizer

Program pro panelizaci postscriptových výstupů z grafických programů jako je Kicad nebo Eagle.

Autor
Mrazík mrazi.nosp@m.k@vo.nosp@m.lny.c.nosp@m.z

Motivace.

Pokud si doma dělám plošný spoj, potřebuji dostat na jednu stranu A4 poněkud více motivů než jen jediný malý obrázek, který je produktem návrhu. Lze to řešit různě - mohu si např. zkopírovat motiv přímo jako celou desku v návrhovém systému a pak teprve generovat postscript. To je ale poněkud otravné (záleží na návrhovém systému), deska se pak dost spatně edituje. Dá se použít OpenOffice, jenže tento program má vlastní interpret postsciptu a ten to obvykle trochu zmrší. Další možnost je udělat to "ručně" pomocí ps kódu např.

%!PS-Adobe-2.0
gsave
x y translate
(soubor1.ps) run
grestore
gsave
x y translate
(soubor2.ps) run
grestore
%...
showpage

Ani to není ideální. Sice to přesně zachová výstupní data, ale je nutné

  • pracně zjišťovat souřednice posunu x, y
  • z původního ps kódu odstranit příkaz showpage

A právě tuto práci by měl vykonat tento program.

Vlastnosti.

  • Program používá pro zobrazení externí balík Ghostscript, který .ps soubor převede do .png a ten je možné pomocí Qt zobrazit.
  • Zatím jsou implementovány pouze translace (přetažení myší) a rotace o 90st. (pravým uchem myši nebo F3).
  • Přidána je změna měřítka kolečkem myši nebo klávesami F1 a F2.
  • Vstup i výstup pouze na A4. Vstup je vlastně dán příkazem Ghostscriptu. Nic moc to neomezuje.

Použití.

Příklad. Tlačítkem (+) přidáme soubor, volně přetáhneme obrázek někam do horního levého rohu, zapneme posun jen v X, tlačítkem (DUP) uděláme kopii, tu přetáhneme vpravo, (DUP) ... atd. Po zaplnění vrchní části uložíme jako nový soubor, tlačítkem (-) postupně smažeme plochu, (+) otevřeme uložený soubor, zapneme posuv Y, (DUP) a posuv dolů ... atd. Nakonec uložit.
